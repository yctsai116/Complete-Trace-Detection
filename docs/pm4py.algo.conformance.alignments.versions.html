pm4py.algo.conformance.alignments.versions.state_equation_a_star module¶

This module contains code that allows us to compute alignments on the basis of a regular A* search on the state-space of the synchronous product net of a trace and a Petri net. The main algorithm follows 1. When running the log-based variant, the code is running in parallel on a trace based level. Furthermore, by default, the code applies heuristic estimation, and prefers those states that have the smallest h-value in case the f-value of two states is equal.

References

1

Sebastiaan J. van Zelst et al., “Tuning Alignment Computation: An Experimental Evaluation”, ATAED@Petri Nets/ACSD 2017: 6-20. `http://ceur-ws.org/Vol-1847/paper01.pdf`_.
